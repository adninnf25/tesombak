FREQUENTLY ASKED QUESTIONS

Pertanyaan umum tentang rank packages

Setelah pembayaran berhasil, staff akan mengaktivasi rank Anda dalam 1-5 menit. Anda akan mendapatkan notifikasi di Discord dan rank akan langsung aktif di server.

Tip: Pastikan Anda online di server saat rank diaktivasi untuk konfirmasi langsung.

Ya, Anda bisa memperpanjang rank sebelum masa aktif habis. Buat ticket di Discord dengan format "Rank Extension" dan pilih metode pembayaran yang sama. Harga extend sama dengan harga pembelian awal.

MEDIA rank diberikan secara gratis untuk content creators (YouTuber, Streamer, TikToker) yang membuat konten tentang server kami. Syaratnya:

  • Minimal 100 subscribers/followers
  • Kontent harus tentang Lucifer Server
  • Minimal 5 menit durasi video/stream
  • Link konten harus dibagikan di Discord

Buat ticket di Discord dengan bukti konten untuk mengajukan MEDIA rank.

Setelah rank habis masa aktifnya, semua fitur eksklusif akan dinonaktifkan. Anda akan kembali ke rank default (member). Namun, nama prefix akan tetap ada sebagai tanda bekas rank holder. Anda bisa memperpanjang kapan saja sebelum atau setelah rank habis.

Jika ada masalah dengan rank Anda (fitur tidak bekerja, rank hilang, dll), segera buat ticket di Discord dengan detail:

  • Username Minecraft Anda
  • Rank yang dibeli
  • Tanggal pembelian
  • Detail masalah yang dialami
  • Screenshot jika ada

Support team akan membantu Anda dalam waktu 24 jam.
